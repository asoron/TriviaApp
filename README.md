## Installation & Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/asoron/TriviaApp.git
   cd trivia-quiz
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the application:
   ```bash
   npm start
   ```

4. Build for production:
   ```bash
   npm run build
   ```
